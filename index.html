Tiny Search Engine

The Tiny Search Engine (TSE) is a program written in C that will take a valid seed URL and allow the user to search for different words on pages reachable from the URL. It then returns a list of pages which contain the relevant word (or words). Keywords like 'and', 'or' and 'not' are also functional when searching for specific word patterns. The program is split into three separate parts: a Crawler, an Indexer, and a Querier.

The Crawler begins by 'crawling' webpages linked to our base URL, and then webpages connected to those, etc. until a maximum search depth is reached. It saves each webpage as a document with its URL and HTML. The Indexer then takes over, using a hashtable implementation to make an index of words, complete with the document they are found in and the frequency with which they are found in that document. The Querier finally connects as a usable client-side interface. It allows users to enter queries for different words or combinations of words, and then will print out the documents in which the words appear along with their frequency.
